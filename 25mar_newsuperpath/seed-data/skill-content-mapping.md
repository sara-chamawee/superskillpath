# Skill x Content Mapping

ไฟล์นี้เก็บข้อมูล mapping ระหว่าง Skill กับ Course content ที่มีอยู่ในระบบ

## โครงสร้างข้อมูล

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| No. | หมายเลขกลุ่ม Skill |
| NEW Domain | หมวดหมู่ใหม่ของ Skill |
| Skill | ชื่อทักษะ |
| Course ID | รหัสคอร์ส |
| Course Name | ชื่อคอร์ส |
| Content Provider | ผู้ให้บริการเนื้อหา |
| Instructor Name | ชื่อผู้สอน |
| Duration | ระยะเวลาคอร์ส |
| To Do List Type | ประเภทการประเมิน (Submit Assignment File / Chat to Assess) |
| To-Do | ลิงก์ไฟล์ To-Do List (ถ้ามี) |

## หมายเหตุ
- แถวที่มี Skill = ชื่อทักษะ เป็นแถวสรุปของกลุ่มคอร์ส
- แถวที่ Skill ว่าง = คอร์สย่อยภายใต้ทักษะนั้น
- To Do List Type มี 2 แบบ: "Submit Assignment File" และ "Chat to Assess"
